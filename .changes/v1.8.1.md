## v1.8.1 - 2026-04-23
### Added
*  `User-Agent` header now includes the library version (e.g. `aura-go-client/v1.8.0`)
### Changed
* Mock types moved from `internal/api` and `internal/httpClient` to `internal/testutil` — mocks no longer compile into production builds
* URL construction now trims stray leading/trailing slashes to prevent double-slash in API endpoint paths
### Fixed
* Removed `StatusStopped` constant whose value was identical to `StatusPaused`, masking genuine "stopped" states
* Added `StatusRestoring` to fix misspelled `StatusRestroying`; deprecated alias retained for backwards compatibility
* Removed unreachable validation guard in `OverwriteFromInstance` — empty-string check above it already returned
* `GetInstanceHealth` no longer stacks two independent timeouts; calls internal `doFetchRawMetrics` directly
* `tenantService.List`, `Get`, and `GetMetrics` now check for a pre-cancelled context, matching all other services
* Removed redundant HTTP 200-status check in `FetchRawMetrics` — the API layer already converts non-2xx to errors
* Duplicate compiled UUID regexes for tenant and snapshot IDs consolidated into a single `uuidRegex`
* `TruncateString` now slices by rune count instead of bytes, preventing truncation mid-multibyte character
### Security
* `CreateInstanceData.String()` added to redact the `Password` field — prevents credentials leaking via logs or fmt.Printf
* WithBaseURL now rejects non-HTTPS URLs to protect OAuth tokens in transit
